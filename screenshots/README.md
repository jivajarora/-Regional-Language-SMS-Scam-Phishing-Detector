# Project Screenshots

Use this folder to store screenshots or GIFs of your interactive web demo. Visual assets make your repository significantly more engaging for interviewers and portfolio reviewers!

## Recommended Screenshots to Add:
1. **Legitimate Message Check**: A screenshot showing a green shield/alert for a legitimate transactional SMS (e.g. OTP or balance credit) with no scam triggers.
2. **Hindi Scam Alert**: A screenshot showing a red alert for a Hindi scam SMS, showing the highlighted triggering words (like `अपडेट` or `तुरंत`).
3. **Hinglish Scam Alert**: A screenshot showing a red alert for a Hinglish scam SMS, highlighting code-mixed scam indicators.
4. **Model Comparison View**: A split view showing baseline predictions vs. transformer predictions to showcase the multi-model architecture.

## How to Embed in your Root README:
Once you save your images in this folder (e.g., `screenshots/demo_scam.png`), you can embed them in your root `README.md` using the following Markdown syntax:

```markdown
![Scam Detection Alert](screenshots/demo_scam.png)
```
